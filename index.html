Algunos de mis trabajos

JavaScript

JavaScript es un lenguaje de programación ampliamente utilizado para el desarrollo web, que permite crear páginas interactivas y dinámicas. Funciona tanto en el lado del cliente como en el servidor a través de entornos como Node.js.

Formatos CSS

CSS (Cascading Style Sheets) es un lenguaje de diseño utilizado para dar estilo y formato a las páginas web. Permite personalizar la apariencia de los elementos HTML, controlando aspectos como colores, fuentes y márgenes.

Flexbox and Grid

Flexbox y CSS Grid son sistemas de diseño en CSS que facilitan la creación de estructuras adaptables en la web, ideales para distintos tamaños de pantalla.
